## 📑 1부: 참/거짓 (T/F) 문항 (10문항)

**[문제] 다음 설명이 맞으면 T, 틀리면 F를 선택하세요.**

1. C언어는 대소문자를 엄격하게 구분하므로, `main`과 `Main`은 전혀 다른 것으로 취급된다.
2. `int` 형 변수에 `3.14`를 저장하면 컴파일 에러가 발생하여 프로그램이 실행되지 않는다.
3. `if` 문의 조건식 안에서 실수로 `a = 5`라고 대입 연산자를 쓰면, 문법 에러(Syntax Error)가 발생한다.
4. `while` 문은 조건식이 처음부터 거짓(0)이면 반복 본문을 단 한 번도 실행하지 않는다.
5. 배열 `int arr[5];`를 선언했을 때, 사용할 수 있는 마지막 원소의 인덱스는 `5`이다.
6. 모든 C 프로그램은 크기와 상관없이 반드시 하나의 `main` 함수를 가져야 한다.
7. `printf` 함수를 사용하여 화면에 출력할 때, 줄바꿈을 하려면 특수문자 `\n`을 사용해야 한다.
8. 관계 연산자(예: `a > b`)의 결과가 참일 때 C언어가 내부적으로 표현하는 숫자는 `1`이다.
9. 포인터 변수는 데이터의 값 자체가 아니라, 그 데이터가 저장된 메모리의 '주소'를 저장하는 변수이다.
10. `//` 기호는 한 줄 주석을 작성할 때 사용하며, 컴파일러는 주석으로 처리된 코드를 실행하지 않는다.

## 📑 2부: 괄호 메우기 (Blank) 문항 (10문항)

**[문제] 빈칸 `(     )`에 들어갈 가장 알맞은 기호, 키워드, 또는 서식 지정자를 적으세요.**

11. 화면에 정수형 변수의 값을 출력하기 위해 `printf` 함수 내에 사용하는 서식 지정자는 `(     )`이다.
12. 변수 `num`이 가진 메모리 상의 '주소값'을 알고 싶을 때 변수명 앞에 붙이는 연산자는 `(     )`이다.
13. 정수 5를 2로 나눈 '나머지' 값인 1을 구하고 싶을 때 사용하는 연산자는 `(     )`이다.
14. 조건문에서 "A와 B가 같다"라는 조건을 표현할 때 사용하는 연산자는 `(     )`이다.
15. `switch` 문에서 어떤 `case`에도 해당하지 않을 때 실행되는 예외 처리용 키워드는 `(     )`이다.
16. 반복문 내부에서 조건에 상관없이 즉시 반복문을 탈출하고 싶을 때 사용하는 키워드는 `(     )`이다.
17. 배열 `int score[10];`이 메모리에서 차지하는 총 크기는 `(     )`바이트이다. (단, `int`는 4바이트로 가정)
18. 문자 `A` 한 글자를 저장하기 위해 사용하는 가장 적절한 기본 자료형 키워드는 `(     )`이다.
19. 열거형을 정의할 때 사용하는 키워드는 `(     )`이며, 멤버에 값을 지정하지 않으면 0부터 자동으로 1씩 증가한다.
20. 함수가 값을 계산한 뒤 호출한 곳으로 결과를 돌려줄 때 사용하는 키워드는 `(     )`이다.

## 📑 3부: 4지선다 (4문1택) 문항 (10문항)

**[문제] 다음 중 가장 올바른 답을 하나만 고르세요.**

21. **다음 중 C언어의 변수명으로 '사용할 수 없는' 것은?**
① `student_id`
② `_score`
③ `3rd_member`
④ `value1`
22. **다음 코드의 실행 결과로 올바른 것은?**
```c
int a = 10, b = 3;
printf("%d", a / b);

```


① `3`
② `3.3333`
③ `3.4`
④ `1`
23. **다음 조건식 중 "변수 `n`이 10보다 크고 20보다 작다"를 올바르게 표현한 것은?**
① `10 < n < 20`
② `10 < n && n < 20`
③ `10 < n || n < 20`
④ `10 < n AND n < 20`
24. **다음 코드의 출력 결과로 올바른 것은?**

```c
    int i = 0;
    while (i < 3) {
        i++;
    }
    printf("%d", i);
    ```
    ① `0`
    ② `2`
    ③ `3`
    ④ `4`

25. **배열 `int arr[] = {10, 20, 30};`이 있을 때, `arr[1]`의 값은 무엇인가?**
    ① `10`
    ② `20`
    ③ `30`
    ④ 에러 발생

26. **다음 중 구조체를 선언할 때 사용하는 키워드로 올바른 것은?**
    ① `union`
    ② `enum`
    ③ `struct`
    ④ `class`

27. **다음 코드의 출력 결과로 올바른 것은?**
    
```c
    int num = 5;
    int *p = &num;
    printf("%d", *p);
    ```
    ① `5`
    ② `num의 주소값`
    ③ `p의 주소값`
    ④ 컴파일 에러

28. **서로 다른 자료형의 멤버들이 '동일한 메모리 공간을 공유'하도록 만드는 사용자 정의 자료형은?**
    ① 구조체(struct)
    ② 공용체(union)
    ③ 열거형(enum)
    ④ 배열(array)

29. **다음 중 무한 루프(종료되지 않고 평생 반복)를 만드는 올바른 문장 구문은?**
    ① `if (1) { }`
    ② `while (0) { }`
    <br>세 가지 표현 중 'less than' 연산자를 피해 단어 형태로 대체하거나 안전하게 작성:
    ③ `for (int i=0; i=10; i++) { }`
    ④ `while (1) { }`

30. **C언어에서 문자열의 끝을 알리기 위해 자동으로 삽입되는 '널 문자(Null terminator)'의 형태로 올바른 것은?**
    ① `\n`
    ② `\t`
    ③ `\0`
    ④ `\b`
